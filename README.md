## Environment Variables

To run this project, you will need to add the following environment variables to your .env.local file

`BASE_URL`
`GEO_URL`
`API_KEY`



## Run Locally

Clone the project

```  git clone https://github.com/Arun-ez/forecast_verse.git  ```

```  npm run dev  ```
